# Todo List Web App

A simple and elegant todo list application built with vanilla JavaScript, HTML, and CSS.

## Features

-  Add, edit, and delete todos
-  Search functionality
-  Filter todos (All, Active, Completed)
-  Dark/Light theme toggle
-  Responsive design
-  Uses DummyJSON API for demo data

## Getting Started

1. Open `index.html` in your web browser
2. Start adding and managing your todos!

## Files

- `index.html` - Main HTML structure
- `main.js` - JavaScript logic and API integration
- `style.css` - Styling and theme definitions

## Technologies Used

- HTML5
- CSS3 (with CSS Variables for theming)
- Vanilla JavaScript (ES6+)
- DummyJSON API

## Usage

- Click the **+** button to add a new todo
- Click on a todo to mark it as complete/incomplete
- Use the search bar to filter todos by text
- Toggle between All/Active/Completed filters
- Switch between dark and light themes with the theme toggle button
